sqlite
简介
sqlite是一个软件库，实现了自给自足、无服务器、零配置的、事务性的sql
数据库引擎。


windows 开发环境搭建
1) http://www.sqlite.org/download.html
请访问 SQLite 下载页面，从 Windows 区下载预编译的二进制文件。
您需要下载 sqlite-shell-win32-*.zip 和 sqlite-dll-win32-*.zip 压缩文件。
创建文件夹 C:\>sqlite，并在此文件夹下解压上面两个压缩文件，
将得到 sqlite3.def、sqlite3.dll 和 sqlite3.exe 文件。
添加 D:\software\sqlite3 到 PATH 环境变量，最后在命令提示符下，使用 sqlite3 命令
