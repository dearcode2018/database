【项目信息】
项目名称 : mysql
中文名称 : 
项目版本 : version 1.0

主要技术 : mysql 、sql语句、分页、函数、

项目描述 : mysql version all

项目备注 : 针对mysql数据库，有针对性地适应于mysql各种语句，和java应用程序对接.
为了对相应数据库的特性和底层sql的探知，数据库项目需要采用jdbc、dbutils、spring jdbcTemplate、mybatis等直接
写sql语句的方式来操作数据库的框架.
将原先[软件模板]中涉及到的模块，可以直接加入到当前数据库项目中，便于查阅和生产应用.

【项目规划】

1) 分页查询
limit


2) 增删改语句


3) 函数调用


4) 存储过程 (mysql 可能还未支持)


5) 语句特性探索


6) 语句优化


查询
1) 搭建环境，通过批量insert制造大量的数据，1百万或几百万行数据
相同数据表结构，不同的表命名，存放不同数量级的数据，1-5百万，到1千万，
分无索引和有索引的情况，使用相同查询语句，分来做验证
2) 创建索引，比较有索引和无索引场景下查询耗时
3) explain 观察查询语句耗时情况
4) 如何优化查询条件(例如where条件顺序)，降低耗时
5) like查询是否使用索引，如何使用
6) 不同类型的数据，如何提高查询效率，例如 数字、字符串、日期/时间、枚举
7) 分组、排序、分页查询
