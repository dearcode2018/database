中文名称 : 
项目版本 : version 1.0

主要技术 : oracle、sql语句、分页、函数、触发器、存储过程、

项目描述 : oracle version 12.x

项目备注 : 针对oracle数据库，有针对性地适应于oracle各种语句，和java应用程序对接.
为了对相应数据库的特性和底层sql的探知，数据库项目需要采用jdbc、dbutils、spring jdbcTemplate、mybatis等直接
写sql语句的方式来操作数据库的框架.
将原先[软件模板]中涉及到的模块，可以直接加入到当前数据库项目中，便于查阅和生产应用.
